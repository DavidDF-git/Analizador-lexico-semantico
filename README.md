# Analizador-lexico-semantico
Proyecto desarrollado para la asignatura de Lenguajes Formales y Autómatas en 4to semestre. Se implementó un analizador léxico-semántico básico en Java, se utilizaron herramientas como JFLEX  y CUP. El analizador esta diseñado para analizar estructuras de repetición en lenguaje C.
